# PureTool - Privacy-First Web Toolbox

PureTool is a collection of useful web tools designed with privacy as the top priority. All processing happens locally in your browserâ€”no data is ever sent to a server.

## Features

- **Image Compressor**: Compress JPG, PNG, and WebP images locally.
- **QR Code Generator**: Create custom QR codes instantly.
- **JSON Formatter**: Prettify and validate JSON data.
- **Password Generator**: Generate secure, random passwords.
- **Dark Mode**: Fully supported dark/light theme switching.
- **PWA Support**: Installable as a standalone app on your device.

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Routing**: React Router DOM

## Development

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment

This project is ready to be deployed on Vercel or Netlify.
